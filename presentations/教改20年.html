異化的心理平衡

如何平衡被異化的感受：

集體認同

把「公司」「團體」當成「我」

那些是「我們公司」的作品

雖然不包含我的設計，也沒關係了

資方：你認同我的而不是你自己的成就，我就可以剝削你了

相對地位認同

把尊貴感建立與別人比較上

跟原本可以合作的人競爭

資方：這樣勞方就不會團結

數字認同

把滿足感建立在數字如「錢」「考績」「成績」上

數字高，就高興了。數字低，就難過了。

證照和學歷也屬此類

資方：操控數字就能操控人，這太容易了

工業時代的教育

國民教育基本是站在資方的立場，目的在實踐：

集體認同

  • 制服、髮禁、學號
  • 齊一課表、齊一進度

把「班級」「學校」當成「我」

那些是「我們班」的作品

雖然不包含我的設計，也沒關係了

資方：你認同我的而不是你自己的成就，我就可以剝削你了

相對地位認同

  • 排名次
  • 個別作業、個別應考而不能合作完成

把尊貴感建立與別人比較上

跟原本可以合作的人競爭

缺少空白時間讓人彼此交誼與自由組織

資方：這樣勞方就不會團結

數字認同

  • 連續的考試
  • 只有評分，沒有分享也沒有用途的作業

把滿足感建立在數字如「成績」「分數」「獎懲」上

數字高，就高興了。數字低，就難過了

證照和學歷也屬此類

資方：操控數字就能操控人，這太容易了

自主工作者

在工業化以前，存在很多自主工作者

擁有不算多但足夠的生產工具

小型工作室

有各自的風格和特色

沒有齊一的規格

創作「作品」而非「產品」

對創作的每個環節，都有參與，也都有概念

Cube - Page - Concave - Zoom - Linear - Fade - None - Default

工業革命下的

    自主工作者受三方面打壓
  • 生產效率：被量產效率下的價格戰打掉
  • 通路管道：被廣播、電視、廣告文宣等行銷戰打掉
  • 政府政策：如圈地政策，直接破壞生產工具與生活空間

資訊革命中的

    自主工作者重新抬頭
  • 生產效率：多人協作平台，增加自主工作者的協作生產力，例如wikipedia和github.com
  • 通路管道：網路行銷與社群網站，增加口碑延燒速度，減低推廣與供需雙方互動成本
  • 政府政策：民間運動更容易組織，政府資訊更透明，形成新的合作與制衡方式，例如g0v.tw

資訊革命中的

    「顧員」被打壓
  • 生產效率：有標準作業流程(SOP)的事，逐步被機器取代。
  • 通路管道：資方連上全球化的人才資源庫，但勞方卻沒有全球公司的資料。資方能換地設廠，勞方很難任意搬家。資訊和機動性的雙重不對等下，能被取代的工作，更容易被取代。
  • 政府政策：未知數

當機器會...

當機器會開車、會講電話、會煎蛋、會栓螺絲與開礦、還會一面飛一面拍照。有SOP的事都會做，世界就不需要「雇員」了。
那時，只剩下「資方」、「自主工作者」和「領失業救濟的人」三種主要的工作形態。

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

Custom Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

Slide Backgrounds

Set data-background="#007777" on a slide to change the full page background to the given color. All CSS color formats are supported.

Image Backgrounds

<section data-background="image.png">

Repeated Image Backgrounds

<section data-background="image.png" data-background-repeat="repeat" data-background-size="100px">

Background Transitions

Pass reveal.js the backgroundTransition: 'slide' config argument to make backgrounds slide rather than fade.

Background Transition Override

You can override background transitions per slide by using data-background-transition="slide".
